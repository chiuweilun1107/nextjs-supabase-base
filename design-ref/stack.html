02 · Database & Auth

Supabase

Postgres, auth, realtime, and storage from one client. Row-level security wired in from the first migration.

  • Email + OAuth auth with refresh-token rotation
  • Migrations managed via the Supabase CLI supabase/migrations
  • Type-safe client generated from schema database.types.ts
  • Realtime channels and broadcast helpers ready to import

03 · Components

shadcn/ui

Components live in your repo, not your node_modules. Read the source, edit a className, ship.

  • 40+ Radix-based primitives, fully themeable
  • Variants typed via cva — no className spaghetti
  • Dark mode controlled by a single class on html.dark
  • Forms wired to react-hook-form + zod from the start

04 · Styling

Tailwind v4

CSS-first configuration, no tailwind.config.js. Tokens live in @theme blocks alongside your CSS.

  • OKLCH color tokens, perceptually-uniform palettes
  • Container queries + cascade layers configured by default
  • Native CSS variables — your brand tokens are real CSS vars
  • 5x faster compiles vs. v3 on a cold cache
