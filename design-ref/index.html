Why this stack.

Three principles guided every dependency choice. Nothing is here by default — every line was earned.

01

Production defaults, not demo code.

Auth, RLS policies, error boundaries, structured logging, and email handlers wired in from the first commit. You start where most projects end up after a month.

02

Owned, not rented.

shadcn/ui means components live in your repo, not a node_modules black box. Read the source, edit one className, ship — no upstream PR, no vendor lock-in.

03

Designed for AI handoff.

A single DESIGN.md is the source of truth for tokens. Pair it with Claude Code and the gap between “wireframe” and “merged PR” collapses to a coffee break.
